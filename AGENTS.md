# LearnCraft AI

## Project Goal

LearnCraft AI transforms topics and documents into structured learning experiences.

The system should generate:

- Lecture-ready presentations
- Study guides
- Deep dives
- Implementation-ready learning content

## Core Principles

- Maximum 3-step workflow
- Selection-based navigation
- No chat-style interaction
- No generic summaries
- Contextual prerequisite injection
- Topic scope detection

## Architecture

Initial Platform:
- Microsoft Copilot Studio
- Azure OpenAI

## Current Phase

Planning & Design

## Priority

1. Product Specification
2. UI Flow
3. Prompt Architecture
4. Copilot Studio Design
5. Implementation
